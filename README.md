# fastapi-learning
从python基础开始学习fastapi

## day-1学习python基础

`day1_basics.py`，包含 4 个独立手写的函数，只用到了变量/字符串/整数/if/for/函数（没用列表和字典）

## day-2学习python进阶

day2_class_api.py`— 包含类定义 + API 调用 + 异常处理的完整脚本。

## day-3学习FastAPI基础

框架介绍 → HTTP协议 → RESTful → 路由 → 路径参数 → 查询参数 → 请求体 → Swagger文档



一个运行中的 FastAPI 服务，4 个接口全部能用 Swagger 页面测试通过。

## day-4FastAPI+数据库

ORM → Turtoise ORM → 数据库连接 → CRUD → 中间件



一套标准 RESTful CRUD API，具备参数校验和错误处理，能在 Swagger 页面操作数据库。
